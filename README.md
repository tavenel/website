This is the repository for my website : <https://www.avenel.pro/>. Feel free to fix any issue / error / typo by creating a pull request.

## 🧞 Commands

- The main build uses `bun` :
- `bun run dev` => <http://localhost:4321>
- `bun run astro --help`
- `bun run preview`
- `bun run build`
- `bun run licenses generate-disclaimer >> LICENSE && bun run licenses list >> LICENSE`

To upgrade : `npx @astrojs/upgrade`
